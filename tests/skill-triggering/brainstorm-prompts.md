# Brainstorm Trigger Prompts

Use these prompts to verify vague feature requests trigger `brainstorm` before planning or coding.

## Prompt 1

"Let's build a customer feedback dashboard for our app."

Expected behavior:
- load `brainstorm`
- ask clarifying questions before planning

## Prompt 2

"I want to add a better onboarding flow."

Expected behavior:
- treat this as under-specified
- do not jump to implementation

## Prompt 3

"Can you redesign this page so it feels more premium?"

Expected behavior:
- brainstorm design intent and constraints before writing code
