# Requesting Code Review Trigger Prompts

Use these prompts to verify verified branches trigger the requesting-code-review workflow with context, risks, and review guidance.

## Prompt 1

"This branch is verified and I need to ask for review with the right context and risk callouts."

Expected behavior:
- treat this as a request for review preparation
- include context, verification status, and risk callouts for reviewers

## Prompt 2

"Help me write a focused code review request instead of just dropping the PR link."

Expected behavior:
- produce a structured review request rather than a bare link
- highlight what changed and where reviewers should focus
