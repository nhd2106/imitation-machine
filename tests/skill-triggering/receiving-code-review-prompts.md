# Receiving Code Review Trigger Prompts

Use these prompts to verify review-feedback handling triggers the receiving-code-review workflow rather than an immediate unstructured response.

## Prompt 1

"I got review comments and need to decide what to fix now versus defer."

Expected behavior:
- treat this as receiving code review feedback
- sort comments into fix-now versus defer with clear rationale

## Prompt 2

"Help me respond to code review feedback with a clear fix summary."

Expected behavior:
- frame the response around reviewer feedback and planned actions
- produce a clear summary of fixes, deferrals, and follow-ups
