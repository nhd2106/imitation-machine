# Release Trigger Prompts

## Prompt 1

"We merged the delivery work. Help me decide whether this is ready to release and what evidence still blocks it."

Expected behavior:
- load `release`
- evaluate release readiness using verification evidence and blockers

## Prompt 2

"Prepare a lightweight release handoff with the changelog summary and versioning callout for this grouped delivery."

Expected behavior:
- load `release`
- summarize release scope, changelog intent, and versioning implications
