# Design Brief Example

## Surface

Portfolio analytics dashboard for design agency founders reviewing weekly client health.

## Direction Lock

- Audience: busy founders scanning between meetings on laptop first, tablet second.
- Visual direction: editorial and confident, with dark neutral framing and a restrained electric accent.
- Memorable anchor: the dashboard should feel like a sharp control room, not a spreadsheet.
- Interaction quality: filters and drill-downs should feel immediate and precise, with calm motion and obvious focus states.

## Responsive And State Notes

- Mobile collapses side panels into a bottom sheet and keeps one primary chart in view.
- Tablet keeps filters visible but reduces comparison density.
- Empty states should coach the user on connecting client data without feeling punitive.

## Handoff Readiness

If implementation starts, preserve the dark editorial framing, keep accent color sparse, and validate the browser result at mobile, tablet, and desktop widths before calling it done.
