# Plan Checklist

## Required per task

- exact file paths
- concrete title and description
- runnable verification command
- expected output
- 2-5 minute estimate

## Reject if present

- TBD / TODO / FIXME
- etc. / and so on / similar to prior task
- commands that require human interpretation
- tasks that combine multiple outcomes

## Approval bar

- implementer should not need to guess
- verifier should know exactly what to run
- task sequence should be dependency-safe
