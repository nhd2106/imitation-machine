# Verification Checklist

Before claiming completion, confirm:

- the full verification command was run recently
- you read the actual output
- you checked failures, not just the exit code
- you are reporting evidence, not confidence

## Allowed completion language

- "`agentic verify all` passed"
- "typecheck passed"
- "tests passed: X pass, 0 fail"

## Disallowed completion language without evidence

- done
- ready
- should work
- looks good
- probably fixed
